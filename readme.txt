Домашнее задание №1 - "Дашборд состояния воздуха в городе"



Установка

1. Сохраните репозиторий
2. Создайте виртуальное окружение "python -m venv my-venv"
3. Активируйте виртуальное окружение "...\my-venv\Scripts\activate"
4. Установите зависимости "pip install -r requirements.txt"
5. Создайте файл ".env" с содержанием:
"
   API_KEY = ваш_ключ
   API_URL = http://api.weatherapi.com/v1/forecast.json
"
6. Запустите программу "python app.py"
7. Откройте дашборд по адресу "http://127.0.0.1:8050/
